# Mapeamento de Indicadores para o climasus4r

Este documento consolida os indicadores identificados no RIPSA/OPAS, na literatura internacional (SVI, SoVI) e nas solicitações do usuário.

## 1. Indicadores Demográficos e Socioeconômicos (Fonte: Censo/IBGE)

| Indicador | Fórmula / Definição | Fonte / Referência |
| :--- | :--- | :--- |
| **Razão de Dependência** | (Pop <15 + Pop >64) / Pop 15-64 | RIPSA / Usuário |
| **Índice de Envelhecimento** | (Pop >64 / Pop <15) * 100 | RIPSA / Usuário |
| **Taxa de Urbanização** | (Pop Urbana / Pop Total) * 100 | RIPSA / Usuário |
| **Densidade Domiciliar** | Média de moradores por dormitório | Usuário (Proxy aglomeração) |
| **Índice de Carência Social** | Composto: % analfabetismo, % sem água, % renda < 1 SM | Usuário |
| **SVI: Socioeconomic Status** | Percentil de: Pobreza, Desemprego, Renda, Sem Diploma | CDC SVI |
| **SVI: Household Comp/Disability** | Percentil de: Idosos, Crianças, Deficiência, Família Monoparental | CDC SVI |
| **SVI: Minority Status/Lang** | Percentil de: Minorias, Proficiência em Inglês (Adaptar para Brasil) | CDC SVI |
| **SVI: Housing Type/Transp** | Percentil de: Multi-unidades, Móveis, Aglomeração, Sem Veículo | CDC SVI |

## 2. Indicadores de Mortalidade (Fonte: SIM + SINASC)

| Indicador | Fórmula / Definição | Fonte / Referência |
| :--- | :--- | :--- |
| **TMI (Infantil)** | (Óbitos < 1 ano / Nascidos Vivos) * 1.000 | RIPSA / Usuário |
| **RMM (Materna)** | (Óbitos maternos / Nascidos Vivos) * 100.000 | RIPSA / Usuário |
| **Mortalidade Prematura DCNT** | Óbitos (Câncer, Diab, Cardiov, Resp) 30-69 anos / Pop 30-69 | ODS / Usuário |
| **Taxa de Homicídios** | (Óbitos CID-10 X85-Y09 / Pop Total) * 100.000 | Usuário |
| **Mortalidade no Trânsito** | (Óbitos CID-10 V01-V99 / Pop Total) * 100.000 | Usuário |
| **Taxa de Suicídio** | (Óbitos CID-10 X60-X84 / Pop Total) * 100.000 | Usuário |
